About This To-Do App

This To-Do List application helps you organize your daily tasks efficiently. You can add new tasks, edit existing ones, mark them as completed, or delete them once done. The goal is to provide a simple yet powerful interface for managing your productivity.

✨ Features

  • User login & authentication
  • Add, update, and delete tasks
  • Mark tasks as completed
  • Search and filter tasks by name or status

💡 Why I Built This

I created this project to improve my skills in Django while learning how to design clean and functional web applications. It also reflects my interest in building tools that make everyday life more organized and efficient.

🛠️ Tech Stack

  • Frontend: HTML/CSS
  • Backend: Django
  • Database: SQLite
